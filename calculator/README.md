# Calculator App

This is a simple calculator application that evaluates mathematical expressions.

## Usage

To run the calculator, execute the `main.py` script with a mathematical expression enclosed in quotes as an argument.

```bash
python main.py "<expression>"
```

## Example

```bash
python main.py "3 + 5 * (10 - 2)"
```

This will output the result of the expression in JSON format.
